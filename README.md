
# FIFA 21 Players EDA
This project performs Exploratory Data Analysis (EDA) on the FIFA 21 players dataset.

## Dataset
The dataset contains information about FIFA 21 players including:
- Age
- Height
- Weight
- Nationality
- Club

## Analysis Performed
- Data loading and inspection
- Checking missing values
- Checking duplicate rows
- Age distribution analysis
- Top nationalities analysis
- Top clubs analysis

## Tools Used
- Python
- Pandas
- Matplotlib
